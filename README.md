bam2base-filter.py: Uses the pysam (v0.23.0) module to align the NGS reads with Q score ⩾30 to the reference sequence and count the nucleotide positions from which the experimental sample deviates from the reference sequence. Input: .bam; Output: .xlsx
bin_fixed_number: Bin mutation rates with a user defined number of bins
bin_fixed_size: Bin mutation rates with a user defined bin size
